`bg-{color}`

Uses the keys defined in `backgroundColor` within the `theme` object of the configuration. Default color palettes can be viewed in the [Palettes](/color/palettes) section.

_Shown here using the `blue` palette_

<Grid>

      />
    </InfoBox>
  ))}
</Grid>
